## tar

- `tar xvf backup.tar`
- `x` extract a tar
- `c` create a new file  
- `v` verbose

## Compress

- `gzip` or `gunzip`

## Example

- `tar cvzf backup.tgz` // compress with gzip
- `tar xvzf backup.tgz` // extract with gunzip
